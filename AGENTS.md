# AGENTS.md

## Repository Goal

本项目研究：
面向 UWSN 的多AUV主动验证与网络恢复机制。

## Development Rules

- 使用模块化设计
- 不允许一次性大规模修改
- 每次只完成一个明确任务
- 所有实验输出 CSV
- 修改代码后必须说明变更文件
- 优先保持代码可运行

## Simulation

平台：
- ns-3
- Aqua-Sim

## Coding Rules

- 所有代码变量名使用英文
- Python >= 3.10
- C++ 代码兼容 ns-3

## Done Criteria

完成任务后：
- 给出运行命令
- 给出输出文件
- 给出日志说明
